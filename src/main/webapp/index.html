A Nossa Biblioteca

A Nossa Biblioteca é uma aplicação de exemplo desenvolvida pela Comunidade de Programadores Java de Angola como um projecto comunitário e actividade de Coworking.

Coworking basicamente se trata de um novo modelo de trabalho que tem o objetivo de incentivar a troca de idéias, compartilhamento, networking e colaboração entre diferentes profissionais que podem ser de diferentes áreas.

Participação e Participantes

Trata-se de um projecto voluntário em que foram convidados todos os membros da comunidade. No entanto a participação é livre e depende da disponibilidade de cada um.

Tecnologias utilizadas

O Projecto visa alicercar as bases de Programação na Plataforma JavaEE

Nesta versão foram utilizadas como tecnologias de base os Servlets e as JSP

Para o desenho das interfaces foi utilizado o Framework Bootstrap3

Para os relatorios foi utilizado o framework JasperReport. Os relatorios foram desenhados com TIBCO Jaspersoft Studio-6.1.0.final
